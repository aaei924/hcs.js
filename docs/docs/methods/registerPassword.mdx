# 비밀번호 설정

비밀번호를 설정합니다.

### 사용법

```ts
const password = "1234" // 설정할 비밀번호

const result = await hcs.registerPassword(school.endpoint, login.token, password)
```

### 결과 예시

[RegisterPasswordResult](/docs/api/RegisterPasswordResult)가 반환됩니다.

```ts
{
  success: true // 성공 여부
}
```
