# 2차 로그인

2차 로그인을 진행합니다.

### 사용법

```ts
const password = "1234"

const secondLogin = await hcs.secondLogin(school.endpoint, login.token, password)
```

### 결과 예시

[SecondLoginResult](/docs/api/SecondLoginResult)가 반환됩니다.

#### 성공
```ts
{
  success: true, // 성공 여부
  token: "Bearer ..." // 2차 로그인 토큰
}
```

#### 실패
```ts
{
  success: false, // 성공 여부
  failCount: 1, // 실패 횟수
  remainingMinutes: 0, // 5번 이상 실패했을 경우 다음 시도까지 남은 시간(분)
  message: "오류 메시지"
}
```
